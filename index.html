Installation Instructions

Windows

  1. Download the .exe installer
  2. Run the installer and follow the setup wizard
  3. Launch Flowtics from your Start menu or desktop shortcut

macOS

  1. Download the .dmg file
  2. Open the .dmg and drag Flowtics to your Applications folder
  3. Launch Flowtics from Applications

Linux

  1. Download the .AppImage file
  2. Make it executable: chmod +x flowtics-*.AppImage
  3. Run the AppImage: ./flowtics-*.AppImage
